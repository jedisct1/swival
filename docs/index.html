A small, powerful CLI coding agent for open models

Swival connects to LM Studio or HuggingFace, sends your task, and runs an autonomous tool-calling loop until it produces an answer. No configuration needed.

$ uv tool install swival
$ swival "Refactor the error handling in src/api.py"

Why Swival

Works with open models

LM Studio auto-discovery, HuggingFace Inference API support. Use whatever model fits your hardware.

Sandboxed by default

File access is scoped to your project directory. Commands are off unless you opt in.

Tool-calling loop

File ops, grep, edit, web fetch, structured thinking. The agent keeps going until the task is done.

Zero config

Install, load a model, go. Swival finds the server and connects automatically.

Quickstart

  1. Install LM Studio and load a model with tool-calling support. Start the LM Studio server.
  2. Install Swival: 
    uv tool install swival
  3. Run: 
    swival "Refactor the error handling in src/api.py"
  4. For interactive sessions: 
    swival --repl
